# Number Wave Pattern

This project prints a special number pattern that forms a wave-like structure.

### Pattern Output
```
For n = 5, 
the output will be:
```

1  
21  
321  
4321  
54321  

### Explanation

- The pattern starts with a single number.
- Each new line increases the count.
- Some lines print numbers in reverse order.
- The pattern looks like a wave when observed diagonally.

### Input

An integer n (number of rows)

### Output

A number pattern with n rows.

### Example

Input:
5

Output:
1
21
321
4321
54321
